# 06. ダッシュボード（Next.js）

## 基本方針

- **自分専用の朝イチダッシュボード** → UIより情報密度優先
- タブで「Apps」「GenAI」を切り替え。将来チャンネルを追加しても拡張しやすい構造
- 認証: Supabase Auth（自分のアカウントのみ）
- デプロイ: Vercel

---

## 画面構成

```
/                          → ダッシュボード（タブで Apps / GenAI を切り替え）
/apps/recommendations      → Apps: AI推薦アプリ全覧（フィルター付き）
/apps/rankings             → Apps: ランキング閲覧
/apps/reports/[id]         → Apps: 分析レポート詳細
/genai/reports/[id]        → GenAI: 動向レポート詳細
/genai/items               → GenAI: 収集アイテム一覧（ソースフィルター付き）
```

## 技術スタック

| 項目 | 技術 |
|------|------|
| フレームワーク | Next.js 14 (App Router) |
| 言語 | TypeScript |
| スタイル | Tailwind CSS + shadcn/ui |
| グラフ | Recharts |
| Markdown表示 | react-markdown |
| 認証 | Supabase Auth |
| データ取得 | Supabase JS SDK (Server Components) |

---

## API Routes

| メソッド | パス | 説明 |
|---------|------|------|
| POST | `/api/apps/analyze` | Apps オンデマンド分析 |
| POST | `/api/genai/analyze` | GenAI オンデマンド分析 |
| GET | `/api/apps/recommendations` | 推薦一覧（フィルター対応） |
| GET | `/api/apps/reports` | Apps レポート一覧 |
| GET | `/api/genai/reports` | GenAI レポート一覧 |
| GET | `/api/genai/items` | GenAI 収集アイテム（フィルター対応） |
| GET | `/api/rankings` | ランキングデータ |
