# 05. AI分析エンジン（Gemini API）

## 概要

**Gemini API（`gemini-2.0-flash`）** を使って各チャンネルの分析を行う。

| 採用理由 | 内容 |
|---------|------|
| 既存キーあり | GEMINI_API_KEY を持っている |
| 無料枠が大きい | 1日1,500リクエスト無料 |
| 速い・安い | 毎日・毎週の自動実行に向いている |
| 長文コンテキスト | ランキングデータを大量に渡せる（1Mトークン対応） |

---

## 📱 Apps チャンネル分析

| 分析タイプ | トリガー | 出力先 |
|-----------|---------|--------|
| 週次市場レポート | GitHub Actions（毎週月曜） | Supabase + Dashboard |
| ギャップ分析 | GitHub Actions（週次） | Supabase + Dashboard |
| オンデマンド推薦 | 「今すぐ分析」ボタン | Dashboard（即時） + Supabase |

---

### 分析 A: Apps 週次レポート

**入力**: 直近7日のランキング変動 + Google Trends スコア変化 + Reddit 人気投稿

**プロンプト**:
```
あなたはスマートフォンアプリ市場のアナリストです。
以下の直近1週間のデータを分析してください。

[ランキングデータ]
{app_rankings の前週比変動上位30件}

[トレンドシグナル]
{trend_signals の直近7日}

以下の観点でレポートを作成してください:
1. 今週急上昇したカテゴリ Top5（理由を数値根拠付きで）
2. 個人開発者が参入できそうなニッチ市場 Top3
3. 需要が高まっているが競合が少ない領域（ギャップ）
4. 来週注目すべきトレンド
```

---

### 分析 B: ギャップ分析

**入力**: 低評価レビュー（1-2星）の頻出不満 + Reddit「こんなアプリ欲しい」スレッド

**プロンプト**:
```
以下は [カテゴリ名] カテゴリのアプリレビューと要望データです。

[低評価レビュー（不満コメント）]
...

[Reddit 要望スレッド]
...

分析してください:
1. ユーザーが既存アプリに感じている不満 Top5
2. 繰り返し出てくる「欲しい機能・体験」
3. 個人開発者が差別化できるポイント
4. このカテゴリで成功するために必要な最低限の機能
```

---

### 分析 C: オンデマンドアプリ提案（核心機能）

**入力**: 直近30日のランキング + トレンドシグナル全体 + 週次レポートサマリー

**プロンプト**:
```
あなたは個人開発者のビジネスアドバイザーです。
以下の市場データを基に、今作るべきスマートフォンアプリを提案してください。

開発者プロフィール:
- スキル: React Native（学習中）, Next.js（経験あり）, Supabase, TypeScript
- 1人開発
- 目標: App Store / Google Play で収益を上げる
- 重要: 「作りたいもの」ではなく「市場が求めるもの」を基準に提案すること

[市場データ]
{ランキング・トレンド・レポートサマリー}

以下の形式で10件提案。ニーズの大きい順。全ゾーン（🟢🟡🔴⬛）を含めること。

---
## 提案 {n}: {アプリ名}

**カテゴリ**: {カテゴリ}
**ターゲットユーザー**: {具体的なユーザー像}
**コンセプト**: {1-2行}
**差別化ポイント**: {既存アプリとの違い}

**市場データ根拠**:
- {具体的な数値: 例「sleep tracker が Google Trends +23%（先週比）」}
- {例「App Store 無料Top20に3本、平均評価2.8★と低い」}

**市場規模**: large / medium / niche
**競合密度**: low / medium / high
**収益化モデル**: freemium / paid / ads / subscription

**実現可能性**: 🟢 NOW / 🟡 LEARN / 🔴 HARD / ⬛ HARDWARE
**判断根拠**: {例: "APIのみで実装可能" / "BLEハードが必要"}
**必要スキル**: {例: React Native, Supabase}
**概算開発期間**: {例: 4〜6週間（1人）} または 見積不可
---
```

## コスト管理

Gemini 2.0 Flash は**無料枠が大きく、分析用途ではほぼコスト0**。

| 分析タイプ | 頻度 | 無料枠内か |
|-----------|------|----------|
| Apps 週次レポート | 週1回（×4/月） | ✅ 無料枠内 |
| ギャップ分析 | 週1回（×4/月） | ✅ 無料枠内 |
| オンデマンド推薦 | 月10〜20回 | ✅ 無料枠内 |
| GenAI 週次レポート | 週1回（×4/月） | ✅ 無料枠内 |
| オンデマンド深掘り | 月10回 | ✅ 無料枠内 |

※ Gemini 2.0 Flash 無料枠: 15 RPM / 1,500 RPD / 1,000,000 TPM（2026年3月時点）
